-------------------------------------java历史-----------------------------
1995年，java语言诞生
1996年，jdk1.0发布
1997年，jdk1.1发布
1998年，java2企业平台j2ee发布
1999年，sun公司发布java的三个版本：标准版，企业版和微型版
2000年，jdk1.3发布
2000年，jdk1.4发布
2001年，j2ee1.3发布
2002年，j2se1.4发布
2004年，j2se1.5发布，成为java语言发展上的一个里程碑
2005年，javase6发布，java的各个版本都已经改名字了，j2se-javaSE，j2ee-javaEE，j2me-javaME
2006年，jre6.0发布
2011年，java7.0发布
2014年，java8.0发布
